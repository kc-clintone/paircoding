# Getting Started with Visual Studio Code (VSCode)

Visual Studio Code (VSCode) is a free, open-source code editor developed by Microsoft that supports multiple programming languages, extensions, and features for modern development.

## Requirements

- **Operating System**: Windows 7+, macOS 10.10+, or Linux (x64 or ARM64)
- **Memory**: At least 1GB of RAM recommended
- **Disk Space**: Minimum 200MB free space

## Installation Steps

1. **Download VSCode**
   - Go to the official download page: [https://code.visualstudio.com/](https://code.visualstudio.com/)
   - Choose the appropriate installer for your operating system.

2. **Install**
   - **Windows**: Run the `.exe` installer and follow the on-screen instructions.
   - **macOS**: Open the `.zip` file and drag `Visual Studio Code` to the Applications folder.
   - **Linux**: Use a deb/rpm package or install via Snap/Flatpak:
     ```
     sudo snap install --classic code
     ```
   
3. **Launch VSCode**
   - Locate the VSCode icon and open the application.

## Basic Setup

1. **Install Extensions**
   - Click the Extensions icon (sidebar or `Ctrl+Shift+X`).
   - Search for popular extensions, like:
     - Python
     - ESLint (JavaScript/TypeScript)
     - Prettier
     - GitLens

2. **Open a Folder or File**
   - Go to `File > Open Folder` or `File > Open File` to start working on your project.

3. **Configure Settings**
   - Access settings via `File > Preferences > Settings` or `Ctrl+,`.
